# JavaScript Calculator — LEXO

A premium calculator built with HTML, CSS and JavaScript.

## Live Demo
[View Calculator](https://lexogamer786-cyber.github.io/javascript-calculator/)

## Features
- Animated gradient border — purple cyan pink
- Ripple effect on every button tap
- Keyboard support — type numbers and operators
- Result flash animation on equals
- Backspace button
- Toggle sign +/−
- Percentage %
- Chain calculations — 5 + 3 × 2
- Divide by zero error handling
- Auto font size — big numbers shrink automatically
- Retro scanline display effect
- Mobile responsive

## Operations
- Addition, Subtraction, Multiplication, Division
- Percentage
- Toggle positive/negative
- Decimal point

## Keyboard Shortcuts
- `0-9` — numbers
- `+ - * /` — operators
- `Enter` or `=` — calculate
- `Backspace` — delete last digit
- `Escape` — clear all
- `%` — percentage

## Tech Used
- HTML5
- CSS3 — gradients, animations, backdrop blur
- JavaScript — all calculator logic
- Google Fonts — Share Tech Mono + Exo 2

## Files
- `calculator.html` — complete single file
